# デザインコンセプト

このドキュメントは、サイトのデザイン方向性とコンセプトを定義します。

---

## 🎨 デザインフィロソフィー

### コアコンセプト

**"ユドナリウムコネクトの世界観を尊重したシンプルで機能的なデザイン"**

ユドナリウムコネクトのブランドカラー（オレンジ → ピンクのグラデーション）と親しみやすい雰囲気を継承しながら、Material Design 的なシンプルさと機能性を両立。開発者がメンテナンスしやすく、ユーザーが直感的に使える UI を実現する。

---

## 🎯 デザイン目標

1. **ブランド継承**: ユドナリウムコネクトの世界観を尊重
2. **シンプル**: 余計な装飾を排除し、情報を明確に
3. **機能的**: Material Design 的な実用性重視
4. **メンテナンス性**: 開発者が理解・更新しやすい構造
5. **親しみやすさ**: 初心者でも安心して使える温かみ

---

## ✅ デザインの実装方針

### 1. カラー戦略

**メインカラー: ユドナリウムコネクトのブランドカラーを継承**

- プライマリ: オレンジ (#FF9966) - ボタン、リンク
- アクセント: ピンク (#E9527E) - CTA、強調要素
- グラデーション: 控えめに使用（ヒーローセクション、見出しなど）

**Material Design の原則:**

- ベース: 白とライトグレーで清潔感
- シャドウ: 軽い影で奥行き表現（過度な装飾は避ける）
- カード: 情報をグループ化、視認性向上

### 2. 形状とレイアウト

**角丸の活用:**

- カード、ボタン: 12-16px の角丸で柔らかい印象
- 画像: 8px の角丸で統一感
- Material Design の elevation（立体感）を適度に使用

**グリッドレイアウト:**

- 8dp グリッドシステム（Material Design 準拠）
- 余白を十分に確保し、情報を整理
- カード型レイアウトで視認性向上

### 3. タイポグラフィ

**シンプルで読みやすく:**

- Noto Sans JP（または M PLUS Rounded 1c）
- 階層を明確に（見出し、本文、補助テキスト）
- 行間を十分に取る

### 4. アニメーション

**控えめで機能的:**

- Material Design のモーションガイドラインに準拠
- ホバー時の軽い変化（0.2-0.3 秒）
- スムーズなページ遷移
- 過度な装飾アニメーションは避ける

### 5. アイコンとビジュアル

**Material Icons + シンプルなイラスト:**

- Material Icons を基本使用
- 必要に応じてシンプルなカスタムアイコン
- 写真とイラストのバランス

## 🎯 具体的な実装ポイント

### やること ✅

1. **ブランドカラーの活用**

   - オレンジ → ピンクのグラデーションを要所で使用
   - プライマリボタン、見出し、アクセント要素

2. **Material Design の原則**

   - シンプルなカードデザイン
   - 適度なエレベーション（影）
   - 明確な視覚階層

3. **丸みと柔らかさ**

   - 12-16px の角丸
   - 優しい影

4. **余白の確保**

   - 情報を詰め込みすぎない
   - セクション間に十分な余白

5. **機能的なアニメーション**
   - ユーザーの操作をサポート
   - フィードバックを明確に

### やらないこと ❌

1. **過度な装飾**

   - 複雑なパターン背景
   - 不要なグラフィック要素
   - 多すぎるアニメーション

2. **メンテナンスしにくい要素**

   - 複雑な CSS トリック
   - カスタムフォント多用
   - 非標準的なコンポーネント

3. **ブランドから外れる表現**
   - 暗い、重い雰囲気
   - 尖った、攻撃的なデザイン
   - 複雑で理解しにくい UI

## 🎭 デザインキーワード

以下のキーワードをデザイン全体で意識する:

1. **Simple（シンプル）**: 余計な装飾を排除、Material Design 的な機能美
2. **Warm（温かい）**: ユドナリウムコネクトの親しみやすさを継承
3. **Functional（機能的）**: ユーザーと開発者双方にとって使いやすい
4. **Maintainable（保守性）**: コードが理解しやすく、更新しやすい
5. **Branded（ブランド）**: ユドナリウムコネクトの世界観を尊重

## 🎨 ビジュアル要素（Material Design 準拠）

### カードデザイン

**ゲームカード（Material Card）:**

- 12px 角丸の白背景カード
- Material Design Elevation 1（軽い影）
- ホバー時: Elevation 2 + 2px 上昇
- ゲーム画像（8px 角丸） + 情報を縦に配置
- パディング: 16px

### アイコン

**使用するアイコンセット:**

- Material Icons（推奨） - Material Design 標準
- Lucide（代替） - シンプルでモダン

**主要アイコン:**

- 👥 Groups（人数）
- ⏱️ Schedule（時間）
- 🏷️ Label（カテゴリー）
- ⭐ Star（お気に入り）
- 🔍 Search（検索）
- 🎮 SportsEsports（ゲーム）

### ボタンスタイル（Material Design 3 種類）

**1. Contained Button（プライマリ）:**

- ブランドグラデーション背景
- 12px 角丸
- 高さ 36px 以上
- Elevation 1 → ホバーで Elevation 2

**2. Outlined Button（セカンダリ）:**

- 透明背景 + 1px ボーダー
- ホバー時に薄い背景色
- 控えめで機能的

**3. Text Button（補助）:**

- 背景なし、テキストのみ
- ホバー時に薄い背景色

### バッジ・タグ（Chip）

**ジャンルタグ:**

- ピルシェイプ（完全な角丸）
- 高さ 24px（Material Design 推奨）
- Material Design カラーで色分け
- 12px フォントサイズ

## 📐 レイアウト原則

### グリッドシステム

**デスクトップ:**

- 12 カラムグリッド
- 最大幅: 1280px
- 左右マージン: 24px

**タブレット:**

- 8 カラムグリッド
- 左右マージン: 20px

**モバイル:**

- 4 カラムグリッド
- 左右マージン: 16px

### スペーシング

**セクション間:**

- 大: 80-120px（デスクトップ）
- 中: 60-80px（タブレット）
- 小: 40-60px（モバイル）

**要素間:**

- 8 の倍数を基準（8px, 16px, 24px, 32px, 40px, 48px）

---

## 🎬 アニメーション原則

### 使用するアニメーション

1. **ページトランジション**: スムーズなフェード
2. **スクロールアニメーション**: 要素が下から上にフェードイン
3. **ホバーエフェクト**: カードが軽く浮き上がる
4. **ローディング**: スケルトンスクリーン

### タイミング

- 早い: 150-200ms（ホバー）
- 標準: 300-400ms（フェード）
- ゆっくり: 500-600ms（スクロール）

### イージング

- `ease-out`: ホバーエフェクト
- `ease-in-out`: ページトランジション
- `cubic-bezier(0.4, 0, 0.2, 1)`: カスタムアニメーション

## 🎯 ユーザビリティ

### アクセシビリティ

- **カラーコントラスト**: WCAG AA 準拠（4.5:1 以上）
- **フォントサイズ**: 本文 16px 以上
- **タップターゲット**: 44×44px 以上
- **キーボード操作**: すべての機能がキーボードで操作可能

### パフォーマンス

- **画像最適化**: WebP 形式、遅延読み込み
- **コード分割**: ページごとに必要なコードのみ
- **初回表示**: 3 秒以内

## 🎪 特徴的な要素（差別化）

### 1. ゲームカードのホバーエフェクト

カードにマウスを乗せると:

- 軽く浮き上がる
- 影が濃くなる
- 「詳細を見る」ボタンが表示される

### 2. ジャンル別カラーコーディング

各ジャンルに固有の色を割り当て:

- 戦略: 🔵 青系
- パーティー: 🟡 黄色系
- 協力: 🟢 緑系
- カード: 🔴 赤系
- etc.

### 3. マイクロインタラクション

- お気に入りボタン: ハートがアニメーション
- フィルター適用: 数字がカウントアップ
- 検索: 入力中にサジェスト表示

### 4. ゲーム詳細ページのギャラリー

- 画像をクリックで拡大
- 左右スワイプで切り替え
- サムネイル一覧
