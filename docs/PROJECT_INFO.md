## 🎮 サイト基本情報

### サイト名

**ユドコネベース**

### サイトの目的

ユドナリウムコネクトで遊べるボードゲームの情報を一元化し、ユーザーが簡単にゲームを探して購入・プレイできるようにする。

### キャッチコピー候補

1. **「ユドナリウムコネクトで、次に遊ぶゲームを見つけよう」**

   - シンプルで行動を促す

2. **「オンラインボードゲームの新しい扉へ」**

   - ワクワク感を演出

3. **「あなたにぴったりのボードゲームが、ここにある」**

   - パーソナライズ感

4. **「探す、見つける、遊ぶ。ユドナリウムコネクトのゲーム図鑑」**
   - 情報サイトとしての役割を明確に

**採用: オプション 1**
「ユドナリウムコネクトで、次に遊ぶゲームを見つけよう」

### サイト説明文（meta description）

「ユドコネベースは、ユドナリウムコネクトで遊べるボードゲームを探せる情報サイトです。ジャンルやプレイ人数で検索して、あなたにぴったりのゲームを見つけよう。」

## 📦 モックデータ仕様

### ゲームデータ構造

```typescript
interface Game {
  id: string; // 一意なID
  slug: string; // URLフレンドリーなID
  title: string; // ゲームタイトル
  titleEn?: string; // 英語タイトル（オプション）
  description: string; // ゲーム説明（短文）
  descriptionLong: string; // 詳細説明
  genre: string[]; // ジャンル（複数可）
  players: {
    min: number;
    max: number;
  };
  playTime: {
    // プレイ時間（分）
    min: number;
    max: number;
  };
  difficulty: "easy" | "normal" | "hard"; // 難易度
  price: {
    min: number;
    max: number;
  };
  images: {
    thumbnail: string; // サムネイル画像URL
    cover: string; // カバー画像URL
    gallery: string[]; // ギャラリー画像
  };
  purchaseLinks: {
    name: string; // 購入先名
    url: string; // 購入URL
  }[];
  tags: string[]; // タグ
  isNew: boolean; // 新着フラグ
  isPickup: boolean; // ピックアップフラグ
  publishedAt: string; // 公開日
}
```

### サンプルゲーム数

- **15-20 個**の架空ボードゲーム
- **7 種類**のジャンル
- **様々なプレイ人数・時間**のバリエーション

2. パーティーゲーム
3. 協力ゲーム
4. カードゲーム
5. 推理ゲーム
6. ファミリーゲーム
7. その他

## 🛒 購入リンク

### 方針

- 特定の EC サイトに誘導しない
- ゲームごとに個別の購入リンクを設定
- 架空の URL でモックアップ作成

### 購入リンク例

```typescript
purchaseLinks: [
  {
    name: "公式サイト",
    url: "https://example.com/game/sample-game",
  },
  {
    name: "Amazon",
    url: "https://example.com/amazon/sample-game",
  },
  {
    name: "ボードゲーム専門店",
    url: "https://example.com/store/sample-game",
  },
];
```
