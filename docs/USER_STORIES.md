# ユーザーストーリー

このドキュメントは、ユドナリウムコネクト対応ゲーム一覧サイトの主要なユーザーストーリーをまとめたものです。

## 主要ペルソナ

### ペルソナ 1: ボードゲーム初心者「田中さん（28 歳）」

- **背景**: オンラインでボードゲームを始めたい
- **目的**: 気軽に遊べるゲームを探したい
- **課題**: どんなゲームがあるのか分からない

### ペルソナ 2: ボードゲーム愛好家「佐藤さん（35 歳）」

- **背景**: 様々なボードゲームをプレイ済み
- **目的**: ユドナリウムコネクトで遊べる新しいゲームを探したい
- **課題**: 対応ゲーム情報が分散していて探しにくい

### ペルソナ 3: ゲームマスター「鈴木さん（42 歳）」

- **背景**: 友人とオンラインゲーム会を主催
- **目的**: グループで楽しめるゲームを効率的に見つけたい
- **課題**: プレイ人数や時間で絞り込みたい

---

## ユーザーストーリー

### Epic 1: ゲーム発見

#### US-1.1 ゲーム一覧の閲覧

**As a** ボードゲームプレイヤー  
**I want to** ユドナリウムコネクトで遊べるゲームの一覧を見たい  
**So that** どんなゲームがあるのか把握できる

**受入基準:**

- ゲーム一覧ページでゲームカードが表示される
- 各カードにタイトル、画像、ジャンル、プレイ人数が表示される
- モバイルでも快適に閲覧できる

#### US-1.2 ゲーム詳細の確認

**As a** ボードゲームプレイヤー  
**I want to** ゲームの詳細情報を見たい  
**So that** そのゲームが自分に合っているか判断できる

**受入基準:**

- ゲーム詳細ページで説明文が読める
- プレイ時間、プレイ人数、難易度が確認できる
- ゲーム画像やルール概要が見られる
- 購入リンクが表示される

#### US-1.3 おすすめゲームの発見

**As a** ボードゲーム初心者  
**I want to** おすすめのゲームを知りたい  
**So that** 何を遊べばいいか迷わない

**受入基準:**

- トップページにピックアップゲームが表示される
- 新着ゲームセクションがある
- カテゴリー別のおすすめが見られる

---

### Epic 2: ゲーム検索

#### US-2.1 キーワード検索

**As a** ボードゲームプレイヤー  
**I want to** ゲーム名で検索したい  
**So that** 特定のゲームを素早く見つけられる

**受入基準:**

- 検索ボックスでゲーム名を入力できる
- 検索結果が即座に表示される
- 該当なしの場合も適切なメッセージが表示される

#### US-2.2 フィルタリング

**As a** ゲームマスター  
**I want to** プレイ人数や時間でゲームを絞り込みたい  
**So that** グループに適したゲームを効率的に見つけられる

**受入基準:**

- ジャンルでフィルタリングできる
- プレイ人数でフィルタリングできる
- プレイ時間でフィルタリングできる
- 複数条件を組み合わせられる

#### US-2.3 ソート機能

**As a** ボードゲーム愛好家  
**I want to** ゲームを並び替えたい  
**So that** 新着順や人気順で確認できる

**受入基準:**

- 新着順でソートできる
- 人気順（将来的）でソートできる
- 名前順でソートできる

---

### Epic 3: 購入サポート

#### US-3.1 購入リンクへのアクセス

**As a** ボードゲームプレイヤー  
**I want to** ゲームを購入できる場所を知りたい  
**So that** すぐに購入できる

**受入基準:**

- ゲーム詳細ページに購入リンクが表示される
- 複数の購入先がある場合は選択できる
- 新しいタブで購入サイトが開く

#### US-3.2 価格情報の確認

**As a** ボードゲームプレイヤー  
**I want to** ゲームの価格を知りたい  
**So that** 予算内で購入できるか判断できる

**受入基準:**

- ゲームカードに価格が表示される
- 価格帯でフィルタリングできる

---

### Epic 4: 情報収集

#### US-4.1 ニュースの閲覧

**As a** ボードゲーム愛好家  
**I want to** 新着情報やアップデート情報を見たい  
**So that** 最新の対応ゲームを知れる

**受入基準:**

- ニュース一覧が表示される
- ニュース詳細が読める
- トップページに最新ニュースが表示される

#### US-4.2 コミュニティへの参加

**As a** ボードゲームプレイヤー  
**I want to** 他のプレイヤーと交流したい  
**So that** 一緒に遊ぶ仲間を見つけられる

**受入基準:**

- Discord サーバーへのリンクが明確に表示される
- サイト内の複数箇所から誘導される

---

### Epic 5: モバイル体験

#### US-5.1 スマートフォンでの閲覧

**As a** モバイルユーザー  
**I want to** スマートフォンで快適にサイトを見たい  
**So that** 外出先でもゲーム情報を確認できる

**受入基準:**

- レスポンシブデザインで表示される
- タップしやすいボタンサイズ
- 読みやすいフォントサイズ

#### US-5.2 高速な表示

**As a** ユーザー  
**I want to** ページがすぐに表示されてほしい  
**So that** ストレスなく閲覧できる

**受入基準:**

- ページ読み込みが 3 秒以内
- 画像が最適化されている
- スムーズなスクロール
